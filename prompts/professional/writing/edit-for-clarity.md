## Purpose
Edit any piece of writing to be clearer, tighter, and more effective for its audience.

## Load Skills
> Before proceeding, read and apply: `skills/writing/SKILL.md`

## Prompt

```
Edit the following [CONTENT TYPE: email / document / report / post] for clarity and impact.

Audience: [WHO WILL READ THIS]
Tone to preserve: [formal / conversational / persuasive / informational]
Priority: [clarity / conciseness / tone / all equally]

[PASTE TEXT HERE]
```

## Usage Notes
- Specify content type — editing a cold email is different from editing a report
- If you want to preserve your voice, say so explicitly in the prompt
- For long documents, specify whether you want section-by-section edits or a full rewrite
