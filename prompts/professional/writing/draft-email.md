## Purpose
Draft a professional email from a brief description of the situation and goal.

## Load Skills
> Before proceeding, read and apply: `skills/writing/SKILL.md`

## Prompt

```
Draft an email for the following situation.

Sender context: [YOUR ROLE / RELATIONSHIP TO RECIPIENT]
Recipient: [WHO THIS IS GOING TO]
Goal: [WHAT THIS EMAIL NEEDS TO ACHIEVE]
Tone: [formal / direct / warm / urgent]
Key points to cover:
- [POINT 1]
- [POINT 2]
- [POINT 3 if needed]

Any context I should know: [BACKGROUND OR SENSITIVITIES]
```

## Usage Notes
- The more context you give about the relationship and goal, the better the output
- If you need multiple versions (e.g. firm vs. softer), say so upfront
- For sensitive situations, describe the dynamic between sender and recipient
