## Purpose
Make sense of a dataset or set of results — identify patterns, anomalies, and meaningful insights.

## Load Skills
> Before proceeding, read and apply: `skills/analysis/SKILL.md`

## Prompt

```
Analyse the following data and identify the key insights.

What this data represents: [DESCRIBE THE SOURCE AND WHAT IT MEASURES]
Time period: [IF RELEVANT]
What I'm trying to understand: [THE QUESTION THIS DATA SHOULD ANSWER]
Known context: [ANYTHING UNUSUAL OR IMPORTANT TO KNOW ABOUT THIS DATA]

[PASTE DATA HERE — table, CSV rows, or summary stats]
```

## Usage Notes
- The clearer you are about the question, the more targeted the insights
- For large datasets, paste a summary or representative sample
- Ask for specific chart recommendations if you need to present the data
