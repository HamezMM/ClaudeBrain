## Purpose
Take a creative idea from vague concept to a defined project with a clear direction and first steps.

## Load Skills
> Before proceeding, read and apply in order:
> 1. `skills/design/SKILL.md` — for creative direction
> 2. `skills/planning/SKILL.md` — for project structure

## Prompt

```
Help me develop the following creative project.

The idea: [DESCRIBE IT — EVEN IF VAGUE OR HALF-FORMED]
Medium or format: [ILLUSTRATION / WRITING / PHOTOGRAPHY / VIDEO / MIXED MEDIA / OTHER / UNSURE]
Purpose: [PERSONAL ENJOYMENT / PORTFOLIO / SHARE PUBLICLY / GIFT / SELL / OTHER]
My current skills in this area: [BEGINNER / INTERMEDIATE / EXPERIENCED]
Time and resources available: [HOW MUCH TIME PER WEEK AND WHAT TOOLS/BUDGET]
What excites me about it: [THE PART YOU DON'T WANT TO LOSE]
What feels unclear: [WHERE YOU'RE STUCK OR UNSURE]
```

## Usage Notes
- "What excites me" and "what feels unclear" together are the most useful inputs — they define what to protect and what to solve
- For projects that cross professional and personal (e.g. a portfolio piece), use the design skill heavily
- Ask for "three different directions this could go" if you want options before committing to one
