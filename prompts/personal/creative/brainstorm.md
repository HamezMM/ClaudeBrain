## Purpose
Generate a wide range of ideas on any creative topic — quantity and variety over polish.

## Prompt

```
Brainstorm ideas for the following.

Topic or challenge: [WHAT YOU'RE TRYING TO GENERATE IDEAS FOR]
Context: [WHY YOU'RE DOING THIS — PERSONAL PROJECT, GIFT, PROBLEM TO SOLVE, ETC]
Constraints: [BUDGET, TIME, SKILLS, TOOLS AVAILABLE — OR NONE]
Quantity: [HOW MANY IDEAS — E.G. 10 QUICK IDEAS / 5 DEVELOPED CONCEPTS / AS MANY AS POSSIBLE]
Direction preference: [PRACTICAL / WILD / BOTH — SHOULD IDEAS BE GROUNDED OR SURPRISING]
What I've already considered: [SO CLAUDE DOESN'T REPEAT THINGS YOU'VE DISMISSED]
```

## Usage Notes
- This is a no-skill prompt by design — brainstorms work best without rigid structure
- "What I've already considered" is very useful — it pushes Claude toward genuinely new territory
- Follow up with "develop idea #3" or "combine ideas 1 and 5" to go deeper on promising ones
- For creative projects, ask for a range across different styles or formats
