## Purpose
Turn a financial goal into a concrete, achievable plan with a timeline and milestones.

## Load Skills
> Before proceeding, read and apply in order:
> 1. `skills/analysis/SKILL.md` — assess feasibility
> 2. `skills/planning/SKILL.md` — build the plan

## Prompt

```
Help me build a plan to achieve the following financial goal.

Goal: [E.G. SAVE $20K FOR HOUSE DEPOSIT / PAY OFF CREDIT CARD / BUILD 6-MONTH EMERGENCY FUND]
Timeframe: [WHEN DO YOU WANT TO ACHIEVE THIS BY, OR "AS SOON AS POSSIBLE"]
Current position: [CURRENT SAVINGS / DEBT BALANCE / STARTING POINT]
Monthly amount available: [HOW MUCH CAN REALISTICALLY GO TOWARD THIS EACH MONTH]
Constraints: [ANYTHING THAT MAKES THIS HARDER — IRREGULAR INCOME, OTHER DEBTS, ETC]
```

## Usage Notes
- If the timeframe and monthly amount don't match the goal, Claude will flag this and suggest options
- Ask for milestone markers (e.g. every $5K saved) if you want progress checkpoints
- Note: Claude is not a financial advisor — treat output as a thinking tool, not professional advice
