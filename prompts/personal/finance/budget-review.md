## Purpose
Review your current spending, identify where money is going, and find meaningful improvements.

## Load Skills
> Before proceeding, read and apply: `skills/analysis/SKILL.md`

## Prompt

```
Review my budget and help me understand where I can improve.

Monthly take-home income: [AMOUNT AFTER TAX]
Financial goal: [SAVE MORE / PAY OFF DEBT / BUILD EMERGENCY FUND / INVEST / GENERAL REVIEW]
Target savings rate or amount: [IF YOU HAVE ONE]

Current monthly spending (fill in what you know):
- Housing (rent/mortgage): 
- Utilities:
- Groceries:
- Eating out / takeaway:
- Transport (car, fuel, public transport):
- Subscriptions:
- Health / gym:
- Entertainment / hobbies:
- Clothing / personal:
- Other regular expenses:

Any context: [RECENT CHANGE IN INCOME, UPCOMING LARGE EXPENSE, DEBT SITUATION]
```

## Usage Notes
- You don't need exact figures — estimates are fine and still produce useful analysis
- Mention any upcoming changes (new job, moving house) so the review accounts for them
- Ask for a "quick wins" list if you want the 3 easiest changes identified separately
- Note: Claude is not a financial advisor — treat output as a thinking tool, not professional advice
