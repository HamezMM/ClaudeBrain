## Purpose
Plan a week of meals around your health goals, dietary needs, and real-life schedule.

## Load Skills
> Before proceeding, read and apply: `skills/planning/SKILL.md`

## Prompt

```
Build me a weekly meal plan based on the following.

Goal: [LOSE WEIGHT / BUILD MUSCLE / EAT HEALTHIER / SAVE MONEY / REDUCE PREP TIME]
Dietary requirements: [VEGETARIAN / VEGAN / GLUTEN-FREE / DAIRY-FREE / NONE / OTHER]
Foods I dislike: [LIST ANYTHING TO AVOID]
Cooking ability and time: [BEGINNER / INTERMEDIATE / ADVANCED — AND MAX PREP TIME PER MEAL]
Meals needed: [BREAKFAST / LUNCH / DINNER / SNACKS — WHICH ONES]
Household size: [HOW MANY PEOPLE]
Budget per week: [APPROXIMATE GROCERY BUDGET]
Meal prep preference: [BATCH COOK ON WEEKENDS / FRESH DAILY / MIXED]
```

## Usage Notes
- Specifying budget and household size makes the plan practical, not just nutritionally ideal
- Ask for a shopping list to be included if you want end-to-end meal prep output
- For weight goals, mention if you want calorie or macro estimates included
