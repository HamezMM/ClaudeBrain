## Purpose
Build a realistic, structured training plan around your goals, schedule, and current fitness level.

## Load Skills
> Before proceeding, read and apply: `skills/planning/SKILL.md`

## Prompt

```
Build me a workout plan based on the following.

Goal: [LOSE WEIGHT / BUILD STRENGTH / IMPROVE ENDURANCE / GENERAL FITNESS / OTHER]
Current fitness level: [BEGINNER / INTERMEDIATE / ADVANCED — AND ANY RELEVANT HISTORY]
Days available per week: [NUMBER AND WHICH DAYS IF FIXED]
Session length: [HOW LONG CAN EACH WORKOUT BE]
Equipment available: [GYM / HOME / BODYWEIGHT ONLY / SPECIFIC EQUIPMENT]
Any injuries or limitations: [BE SPECIFIC — E.G. BAD KNEES, LOWER BACK ISSUES]
Anything I enjoy or hate: [E.G. LOVE LIFTING, HATE RUNNING]
```

## Usage Notes
- Be honest about current fitness level — overestimating leads to a plan you won't stick to
- If you have an existing routine, describe it so the plan can build on or replace it sensibly
- Ask for a progressive version (weeks 1-4, then 5-8) if you want periodisation built in
