## Purpose
Build a structured learning path for any skill — from zero to a defined level of competency.

## Load Skills
> Before proceeding, read and apply in order:
> 1. `skills/research/SKILL.md` — understand the skill landscape
> 2. `skills/planning/SKILL.md` — build the learning plan

## Prompt

```
Help me learn the following skill.

Skill: [WHAT YOU WANT TO LEARN]
Current level: [COMPLETE BEGINNER / SOME EXPOSURE / INTERMEDIATE WANTING TO ADVANCE]
Target level: [WHAT DOES "GOOD ENOUGH" LOOK LIKE FOR YOU]
Time available: [HOURS PER WEEK AND ROUGH TIMEFRAME]
Learning style preference: [READING / WATCHING / DOING / PROJECTS / MIXED]
Resources I already have: [BOOKS, COURSES, TOOLS — OR NONE]
```

## Usage Notes
- Defining "target level" is the most important input — "I want to be good" is less useful than "I want to build a personal website"
- Ask for a resource list alongside the plan if you want specific recommendations
- For creative skills, ask for a project-based plan rather than a theory-first one
