## Purpose
Learn about any topic thoroughly — from a structured overview to a deep dive ready for action.

## Load Skills
> Before proceeding, read and apply: `skills/research/SKILL.md`

## Prompt

```
Research the following topic for me.

Topic: [WHAT YOU WANT TO UNDERSTAND]
Why I'm learning this: [CONTEXT — PERSONAL INTEREST / DECISION TO MAKE / BUILDING SOMETHING]
What I already know: [YOUR CURRENT LEVEL — NONE / BASICS / INTERMEDIATE]
Depth needed: [OVERVIEW / THOROUGH / AS DEEP AS POSSIBLE]
Anything specific to focus on: [IF YOU HAVE A SPECIFIC ANGLE OR QUESTION]
```

## Usage Notes
- "Why I'm learning this" shapes the output significantly — learning for fun vs. making a decision are very different
- For technical topics, state your background so the level of explanation is right
- Follow up with "go deeper on [section]" to expand any part of the output
