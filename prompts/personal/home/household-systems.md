## Purpose
Design a simple, repeatable household system or routine that actually gets followed.

## Load Skills
> Before proceeding, read and apply: `skills/home-admin/SKILL.md`

## Prompt

```
Help me build a household system for the following.

What I want to systematise: [E.G. WEEKLY CLEANING / GROCERY SHOPPING / BILL MANAGEMENT / MAINTENANCE SCHEDULE]
Household: [HOW MANY PEOPLE, ANY KIDS OR PETS]
Current situation: [WHAT HAPPENS NOW — EVEN IF "NOTHING STRUCTURED"]
Pain point: [WHAT KEEPS GOING WRONG OR GETTING MISSED]
Time available: [HOW MUCH TIME REALISTICALLY PER WEEK FOR THIS]
Preference: [DETAILED CHECKLIST / SIMPLE ROUTINE / DIGITAL / PAPER]
```

## Usage Notes
- The "pain point" field is the most important — it tells Claude what the system actually needs to solve
- Ask for a seasonal version if the task has annual cycles (e.g. maintenance schedules)
- For shared households, ask Claude to include an ownership column so tasks have a clear person
