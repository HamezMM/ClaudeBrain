## Purpose
Plan a home improvement or renovation project with phases, sequencing, and a reality check on budget and time.

## Load Skills
> Before proceeding, read and apply in order:
> 1. `skills/home-admin/SKILL.md` — for home project structure
> 2. `skills/planning/SKILL.md` — for the plan format

## Prompt

```
Help me plan the following home project.

Project: [WHAT YOU WANT TO DO — E.G. RENOVATE KITCHEN, REPAINT LIVING ROOM, ADD STORAGE]
Property type: [OWNED / RENTED — AND HOUSE / APARTMENT / OTHER]
Current state: [DESCRIBE WHAT EXISTS NOW]
Goal state: [WHAT DONE LOOKS LIKE]
DIY or professional: [FULL DIY / SOME DIY / ALL PROFESSIONAL]
Budget: [TARGET RANGE OR "HELP ME ESTIMATE"]
Timeline: [WHEN YOU WANT IT DONE BY OR "FLEXIBLE"]
Constraints: [ANYTHING THAT COMPLICATES THIS — LIVING IN SPACE DURING WORKS, SHARED BUILDING, ETC]
```

## Usage Notes
- "Rented" changes the output significantly — mention it so the plan stays within what's permitted
- If budget is unknown, Claude will provide a rough range based on the project scope
- Ask for a contractor brief to be included if you'll be getting quotes
